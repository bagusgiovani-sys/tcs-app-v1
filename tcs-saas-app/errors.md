# Error Log — TCS Coffee App
> Append-only. Never edit existing entries. Never delete.
> Read this at the start of every Claude Code session to avoid repeating mistakes.

## Entry Format

---
## [YYYY-MM-DD HH:MM] [Short error title]
#[type] #[concept]
**Context:** What were you trying to build when this happened
**Error:** The exact error message or description of what went wrong
**Root cause:** Why it happened — dig to the actual reason, not just the symptom
**Fix:** Exactly what was done to resolve it
**Prevention:** What rule should be added to [[CLAUDE.md#What NOT To Do]] to prevent this next time
**Files affected:** List of files changed as part of the fix
**Related:** [[CLAUDE.md#relevant-section]]
---

## Rules

- Every error gets an entry, no matter how small or embarrassing
- **Prevention is the most important field** — extract the lesson, not just the fix
- If the same error appears a second time, that is a CLAUDE.md failure — update "What NOT To Do" immediately
- Append at the bottom, never edit existing entries
- Read this before every session alongside CLAUDE.md and build-progress.md
